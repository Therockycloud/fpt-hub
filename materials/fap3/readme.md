## Ảnh 1

Trang chính của trường cung cấp link tải SEB và EOS, trong đó SEB dùng
cho bài kiểm tra nhỏ và EOS dùng cho các kỳ thi chính, nếu link chính
lỗi thì sinh viên dùng bản backup.

## Ảnh 2

Hiển thị thông tin tải và nâng cấp SEB, EOS Client và bản backup để sinh
viên sử dụng khi thi.

## Ảnh 3

Hướng dẫn tải SEB trên Google Drive, chia thư mục theo Windows và
MacBook kèm file hướng dẫn.

## Ảnh 4

Hướng dẫn cài đặt SEB trên macOS bằng file .pkg và tài liệu đi kèm.

## Ảnh 5

Hoàn tất cài đặt SEB trên macOS trong thư mục Download và sử dụng sau
khi cài thành công.

## Ảnh 6

Hướng dẫn cài đặt SEB trên Windows bằng video và file zip.

## Ảnh 7

Giải nén và cài đặt SEB trên Windows bằng Extract All và chạy file cài
đặt.

## Ảnh 8

Sử dụng công cụ Microsoft Program Install and Uninstall để sửa lỗi SEB.

## Ảnh 9

Gỡ SEB bằng Control Panel và xóa cache nếu cần.

## Ảnh 10

Khắc phục lỗi Start Session Error bằng cách cài lại Visual C++.

## Ảnh 11

Sửa lỗi Installations cannot be found và Start Session Error bằng Device
Manager và công cụ hỗ trợ.

## Ảnh 12

hình.

## Ảnh 13

## Ảnh 14

## Ảnh 15

## Ảnh 16

## Ảnh 17

## Ảnh 18

## Ảnh 19

## Ảnh 20

## Ảnh 21

## Ảnh 23
