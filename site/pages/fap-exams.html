1. Tải & Cài đặt SEB/EOS

ℹ️ SEB vs EOS
SEB (Safe Exam Browser): Dùng cho các bài kiểm tra nhỏ (Quiz, PE)
EOS Client: Dùng cho kỳ thi chính (Final Exam)

Link tải chính thức

Trang FAP cung cấp link tải SEB và EOS. Nếu link chính bị lỗi, dùng bản backup.

Cài đặt trên macOS

Tải file .pkg từ Google Drive → thư mục MacBook → chạy cài đặt → hoàn tất trong Download.

Cài đặt trên Windows

Giải nén file zip bằng Extract All → chạy file cài đặt.

Cài đặt EOS Client

Giải nén và chạy Run as Administrator.

2. Xử lý lỗi phần mềm

Lỗi Start Session Error

Cài lại Visual C++ Redistributable để khắc phục lỗi này.

Lỗi Prohibited Display Configuration

Gỡ driver màn hình trong Device Manager để khắc phục.

Gỡ và cài lại SEB

Dùng Microsoft Program Install and Uninstall hoặc Control Panel để gỡ hoàn toàn, xóa cache nếu cần.

3. Chuẩn bị trước khi thi

Checklist chuẩn bị

✅ Máy Windows, phần mềm mới nhất
✅ Kết nối WiFi FPT_EXAMONLINE
✅ Tai nghe (cho bài thi Listening)
✅ Font chữ, bộ gõ tiếng Việt
✅ Quyền Administrator

4. Quy chế phòng thi

• Đến đúng giờ với giấy tờ tùy thân
• Giữ trật tự, tuân thủ nội quy
• Không gian lận — vi phạm sẽ bị kỷ luật

⚠️ Lưu ý nghiêm trọng
Mọi hành vi gian lận sẽ bị ghi nhận và xử lý kỷ luật theo quy chế nhà trường.

5. Xử lý sự cố trong khi thi

• Nếu máy bị treo: tắt máy và báo giám thị
• Lỗi đăng nhập / mạng / exam code: kiểm tra kết nối hoặc báo giám thị
• Không chọn được đáp án: điều chỉnh scale màn hình và bộ gõ

6. Phúc khảo bài thi

Gửi đơn phúc khảo

Truy cập FPT Academic Portal → chọn Send Application → điền đầy đủ thông tin phúc khảo.
